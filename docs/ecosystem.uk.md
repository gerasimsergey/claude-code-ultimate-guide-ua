# Екосистема: 4 взаємопов'язані репозиторії

Цей посібник є частиною екосистеми з 4 репозиторіїв, що розділяють аудиторії (розробники проти офісних працівників) та сценарії використання (документація проти демонстрації).

## Архітектура

```
        РЕПОЗИТОРІЇ-ДЖЕРЕЛА (Документація)
        ┌──────────────────┬──────────────────┐
        │                  │                  │
    ┌───▼───┐          ┌───▼───┐
    │ Guide │          │Cowork │
    │ Code  │◄────────►│ Guide │
    │ vX.Y  │ посилання│ v1.0  │
    └───┬───┘          └───┬───┘
        │                  │
        │ джерело          │ джерело
        │                  │
        ЛЕНДИНГИ (Демонстрація)        РОЗПОДІЛ
        ┌──────────────────┬──────────────────┐       ┌──────────────┐
        │                  │                  │       │              │
    ┌───▼───┐          ┌───▼───┐          ┌───▼───────▼──┐
    │ Code  │◄────────►│Cowork │          │   Маркетплейс│
    │Landing│ посилання│Landing│          │   плагінів   │
    │ vX.Y  │          │ v1.0  │          │ (8 плагінів) │
    └───────┘          └───────┘          └──────────────┘
```

## 1. Claude Code Ultimate Guide (цей репозиторій)

**Для**: Розробників, які використовують Claude Code CLI.
**Зміст**: Посібник на 25K+ рядків, 235 шаблонів, воркфлоу, архітектура.

## 2. Claude Cowork Guide (окремий репозиторій)

**Для**: Офісних працівників, які використовують Claude Desktop (не розробники).
**Зміст**: 6 посібників, 67 промптів, 5 воркфлоу, шпаргалка, FAQ.
**Примітка**: Папку `cowork/` було перенесено з основного репозиторію в цей окремий (v1.0.0).

## 3. Лендинг Code Guide

**Призначення**: Сторінка для відвідувачів, які знайомляться з посібником для розробників.
**Зміст**: Маркетингова сторінка, бейджі, FAQ, квіз (334 питання).

## 3b. Плагіни Claude Code (Маркетплейс)

**Для**: Розробників, які хочуть встановлювати шаблони посібника без ручного копіювання файлів.
**Зміст**: 8 плагінів, 181 шаблон. **Першоджерело знаходиться тут**, у папці `examples/`.

**Встановлення**:
```bash
claude plugin marketplace add FlorianBruniaux/claude-code-plugins
claude plugin install security-suite   # або будь-який з 8 плагінів
```

## 4. Лендинг Cowork Guide

**Призначення**: Сторінка для відвідувачів, які знайомляться з посібником для не-розробників (Cowork).

## Синхронізація між репозиторіями

| Зміна | Репозиторії для оновлення |
|--------|----------------|
| Нова версія (Code Guide) | 1. Code Guide, 2. Code Landing |
| Додавання/видалення шаблонів | 1. Code Guide, 2. Code Landing, 3. Plugins repo |
| Додавання/оновлення плагіна | 1. Plugins repo, 2. Code Guide README, 3. Landing |
| Нова версія (Cowork) | 1. Cowork Guide, 2. Cowork Landing |
| Додавання/видалення промптів | 1. Cowork Guide, 2. Cowork Landing |

## Принцип
Чітке розділення аудиторій, легка навігація та підтримання синхронізації між усіма частинами проекту.
