**Помилки з'єднання: поширені причини**

| Помилка | Причина | Рішення |
|---------|---------|---------|
| `ECONNREFUSED` | Сервер не запущений | Перевірте правильність команди в `mcp.json` |
| `Timeout after 30s` | Повільна ініціалізація | Збільште таймаут або перевірте логи сервера |
| `Module not found` | Відсутні залежності | Запустіть `npm install` у директорії сервера |
| `ENOENT` | Бінарний файл не знайдено | Перевірте, чи є npx/npm у вашому PATH |

### Проблеми з правами доступу (Permissions)

**Помилки в шаблонах (Pattern matching)**:
- **❌ Погано**: `"allowedTools": ["Bash(npm test)"]`. Це дозволить ТІЛЬКИ команду `npm test` і нічого більше.
- **✅ Добре**: `"allowedTools": ["Bash(npm *)"]`. Використовуйте зірочку для дозволу будь-яких команд npm.

**Рекомендований набір дозволів**:
Додайте в `settings.json` дозволи для `git *`, `npm *`, `Edit`, `Write` та інструментів ваших MCP-серверів (`mcp__serena__*`). Це значно прискорить роботу, оскільки Claude не запитуватиме дозвіл на кожну дрібну дію.

### 10.5 Claude Code Cheatsheet (Шпаргалка)

**Короткий довідник на одну сторінку**:
- **Команди**: `/help`, `/clear`, `/status`, `/compact`, `/plan`, `/rewind`.
- **Швидкі дії**: `!команда` (термінал), `@файл` (посилання), `Ctrl+R` (пошук).
- **Контекст**: 0-50% (вільно), 75-90% (`/compact` негайно), 90%+ (`/clear` обов'язково).
- **Thinking Mode**: `Alt+T` для швидкого перемикання глибини міркувань.
