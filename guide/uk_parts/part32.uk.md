### Додаток B: FAQ (Часті питання)

**Claude Code vs ClawdBot: У чому різниця?**
- **Claude Code**: Для розробників у терміналі. Написання коду, рефакторинг, архітектура.
- **ClawdBot**: Персональний асистент у месенджерах (WhatsApp/Telegram). Розумний дім, нагадування, щоденні завдання.
**Висновок**: Це не конкуренти, а взаємодоповнюючі інструменти.

### Додаток D: Міфи проти Реальності

**❌ Міф**: "Claude Code має секретні прапорці, що відкривають приховані функції".
**✅ Реальність**: Усі функції задокументовані в офіційному CHANGELOG. "Секрети" — це зазвичай експериментальні функції (як TeammateTool), які ще нестабільні.

**❌ Міф**: "Claude Code — це повна заміна Junior-розробників".
**✅ Реальність**: Це підсилювач навичок. Він потребує кваліфікованого наглядача, який розуміє системний дизайн та проводить верифікацію результатів.
