## Як використовувати цей шлях

### Крок 1: Оцініть свій рівень

**Початківець** (день 1 з Claude Code):
- Пройдіть швидкий шлях: 01 → 02 → 03 (2 години).
- Потім перейдіть до повного посібника `Частина 1: Швидкий старт`.

**Середній рівень** (використовуєте Claude Code 1-4 тижні):
- Пройдіть повний шлях: 01 → 07 (8-11 годин).
- Сфокусуйтеся на модулях 04, 05, 06, 07 для глибини.

**Просунутий рівень** (використовуєте Claude Code 1+ місяць):
- Швидко перегляньте 01-03, детально вивчіть 04-07.
- Перейдіть до Частини 9 посібника: Просунуті патерни оркестрації.

### Крок 2: Оберіть свій трек

**Трек A: Майстер основ** (11 годин)
- Слідуйте модулям 01-07 по порядку.
- Виконуйте всі вправи.
- Використовуйте `/lesson-quiz [topic]` після кожного модуля.
- Завершіть за допомогою `/self-assessment comprehensive`.

**Трек B: Спеціалізоване занурення** (4-6 годин)
- Модулі 01-02 (основи).
- Перейдіть до модулів 04, 05 або 06 залежно від вашої мети.
- Вивчіть обраний модуль детально.

**Трек C: Впровадження в команду** (5-7 годин)
- Модулі 01-02 (навчання основ).
- Модуль 03 (командна конфігурація).
- Модулі 04-07 (командні воркфлоу).
- Запустіть `/self-assessment` для всієї команди, щоб виявити прогалини.

## Деталі модулів

### Модуль 01: Встановлення та налаштування (15 хв)
**Мета:** Запустити Claude Code та підтвердити його роботу.
- Встановлення для вашої платформи (macOS/Linux/Windows).
- Запуск першої команди.
- Розуміння циклу "запит → відповідь".
- **Вправа:** Виконайте воркфлоу "Перші 5 хвилин".

### Модуль 02: Основний цикл (45 хв)
**Мета:** Зрозуміти, як Claude Code працює на системному рівні.
- Цикл взаємодії (запит → інструмент → рішення → фідбек).
- Контекст та сесії.
- Плани та режими мислення.
- **Вправа:** Використайте `/plan` для реального завдання, спостерігайте за процесом.

### Модуль 03: Пам'ять та конфігурація (1 год)
**Мета:** Налаштувати Claude Code під свій воркфлоу.
- Структура та ієрархія `CLAUDE.md`.
- Пріоритетність налаштувань (global → project → session).
- Змінні оточення.
- **Вправа:** Створіть свій перший `CLAUDE.md`, персоналізуйте налаштування.

### Модуль 04: Агенти та спеціалізація (1.5 год)
**Мета:** Створювати фокусованих агентів для конкретних завдань.
- Коли використовувати агентів замість однієї сесії.
- Обмеження та ізоляція інструментів.
- Створення агентів (формат `AGENT.md`).
- **Вправа:** Створіть 2 агенти (універсал + спеціаліст), протестуйте їх.

### Модуль 05: Скіли та автоматизація (1.5 год)
**Мета:** Створювати багаторазові можливості, що запускаються автоматично.
- Життєвий цикл скіла та frontmatter.
- Патерни авто-виклику.
- Об'єднання скриптів зі скілами.
- **Вправа:** Створіть скіл, який вирішує повторювану проблему.

### Модуль 06: Хуки та події (1 год)
**Мета:** Автоматизувати відповіді на системні події.
- Події хуків та умови спрацювання.
- Попередня валідація vs виконання після завершення.
- Написання безпечних хуків.
- **Вправа:** Додайте 2 хуки (пре- та пост-) до вашої конфігурації.

### Модуль 07: Просунуті патерни (2-3 год)
**Мета:** Оркестрація мульти-агентних воркфлоу.
- Мульти-агентні команди та залежності.
- Патерни оркестрації.
- Обробка помилок та відновлення.
- **Вправа:** Спроектуйте воркфлоу з 3-х агентів для реалістичного сценарію.
