# Керування командними ШІ-інструкціями

Керуйте інструкціями для ШІ (`CLAUDE.md`, `.cursorrules`) у великій команді без ризику фрагментації.

**Патерн**: Модульна збірка на основі профілів (Profile-Based Module Assembly) — спільні модулі + персональні профілі розробників + автоматизований скрипт збірки.

**Коли використовувати**: Команда від 5 розробників, використання різних ШІ-інструментів (Claude Code + Cursor/Windsurf), різні ОС (macOS, Linux).

---

## 1. Проблема фрагментації (N x M x P)

Коли команда росте, інструкції фрагментуються:
- **N розробників** (5-20 осіб)
- **M інструментів** (Claude Code, Cursor, Copilot)
- **P операційних систем** (macOS, Linux, WSL)

**Результат**: 30+ можливих конфігурацій. Без системи кожен розробник має свою версію `CLAUDE.md`, і ніхто не знає, яка з них є актуальною.

---

## 2. Архітектура рішення

Замість одного монолітного файлу ми використовуємо **композицію**:

- **Модулі (`modules/`)**: Фрагменти Markdown (стандарти TypeScript, воркфлоу Git, шляхи для macOS).
- **Профілі (`profiles/`)**: YAML-файли розробників, де вказано їхні ОС, інструменти та набір потрібних модулів.
- **Скелет (`skeleton/`)**: Шаблон з плейсхолдерами `{{MODULE:name}}`.
- **Асемблер (`sync-ai-instructions.ts`)**: Скрипт, який читає профіль, підставляє модулі в скелет і видає готовий `CLAUDE.md`.

---

## 3. Фази впровадження

1. **Аудит**: Класифікуйте кожний рядок вашого поточного `CLAUDE.md` як Універсальний, Умовний (залежить від ОС) або Персональний.
2. **Екстракція модулів**: Створіть окремі `.md` файли для кожної тематичної групи (наприклад, `typescript-rules.md`, `git-workflow.md`).
3. **Створення профілів**: Кожен розробник створює свій YAML-файл зі списком модулів.
4. **Автоматизація**: Налаштуйте скрипт збірки та CI-перевірку на розбіжності (Drift Detection), щоб гарантувати, що розробники не редагують згенеровані файли вручну.

---

## 4. Онбординг за 5 хвилин

Новий розробник просто копіює шаблон профілю, вказує свою ОС та інструменти, запускає скрипт і отримує ідеально налаштований `CLAUDE.md` під свій робочий стіл.
