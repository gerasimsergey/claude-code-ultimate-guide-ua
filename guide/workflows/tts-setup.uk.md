# Налаштування TTS — встановлення Agent Vibes

**Мета**: Додати функцію озвучування відповідей (Text-to-Speech) до Claude Code
**Час**: ~18 хвилин
**Складність**: Середня
**Система**: macOS (потрібен Homebrew)

---

## 1. Чи варто встановлювати TTS?

Швидка оцінка:
- Працюєте з довгими рев'ю коду? (+2)
- Виконуєте кілька завдань одночасно під час дебагу? (+2)
- Потрібен офлайн-голос (без хмари)? (+2)
- Працюєте в публічних місцях без навушників? (-3)
- Віддаєте перевагу тиші? (-2)

**Результат**: ≥3 — встановлюйте; <0 — краще пропустити.

---

## 2. Огляд етапів

1. **Підготовка** (5 хв): Встановлення Bash 5.x та аудіо-інструментів.
2. **Інсталяція Agent Vibes** (5 хв): Запуск `npx agentvibes install`.
3. **Piper TTS** (5 хв): Налаштування якісних офлайн-голосів.
4. **Тестування** (3 хв): Перевірка звуку та конфігурація.

---

## 3. Ключові команди

Встановлення залежностей:
```bash
brew install bash sox ffmpeg espeak-ng
```

Запуск інсталятора:
```bash
npx agentvibes install
```
*Порада: Обирайте `Piper TTS` як провайдера для найкращої якості звуку без затримок.*

---

## 4. Оптимізація роботи

- **Для рев'ю коду**: Встановіть `/agent-vibes:verbosity low`, щоб Claude був менш "балакучим".
- **Економія батареї**: Використовуйте команду `/agent-vibes:provider switch macos` для перемикання на системний голос Apple, який майже не споживає ресурси процесора.
- **Тимчасовий режим тиші**: Команда `/agent-vibes:mute` вимикає голос без видалення програми.
