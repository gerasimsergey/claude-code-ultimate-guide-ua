# Глосарій Claude Code

Алфавітний довідник термінів, які ви зустрінете в цьому посібнику. Охоплює специфічні концепції Claude Code, патерни, створені спільнотою, та словник ШІ-інженерії. Стандартні терміни CS/DevOps (JWT, CI/CD, REST) сюди не входять — шукайте їх в інших джерелах.

**Формат**: Термін | Визначення | Категорія | Підкатегорія
