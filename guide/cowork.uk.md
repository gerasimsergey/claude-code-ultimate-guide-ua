# Claude Cowork: Агентний десктоп для інтелектуальної праці

> **📦 Повна документація перенесена в окремий репозиторій**

## Швидкий огляд

**Cowork** — це функція агентного десктопа Claude, яка розширює автономні можливості ШІ для нетехнічних користувачів через додаток Claude Desktop. Замість термінальних команд, Cowork отримує доступ до локальних папок і файлів безпосередньо.

### Ключові факти

| Аспект | Деталі |
|--------|---------|
| **Статус** | Дослідницьке прев'ю (Research preview, січень 2026) |
| **Доступ** | Підписка Pro ($20/міс) або Max ($100-200/міс), лише macOS |
| **Фокус** | Маніпуляція файлами, генерація документів, організація |
| **Відмінність від Code** | Не виконує код — лише робота з файлами |

---

## Три інструменти Claude: Який обрати?

Три інструменти, одна підписка ($20/міс Pro). Вони доповнюють один одного, а не конкурують.

| | Claude AI | Claude Code | Cowork |
|---|-----------|-------------|--------|
| **Інтерфейс** | Веб / Мобільний | Термінал (CLI) | Десктопний додаток |
| **Слоган** | Ви пишете, думаєте, шукаєте | Ви кодите масштабно | Ви автоматизуєте без коду |
| **Визначення** | Універсальний розмовний асистент | Автономний агент для цілих кодобаз | Агентні воркфлоу для не-девелоперів |
| **Основні сценарії** | Тексти, брейнштормінг, дослідження | Дебаг, рефакторинг, тестування | Організація файлів, витяг з PDF |
| **Виконує код** | Ні | Так | Ні |
| **Доступ до файлів** | Лише завантаження | Повний | Пісочниця (папка) |
| **Налаштування** | Не потрібне | npm install -g @anthropic-ai/claude-code | Встановлення додатка macOS |
| **Зрілість** | Продакшн | Продакшн | Дослідницьке прев'ю |
| **Ідеальний профіль** | Письменник · Консультант · Студент | Розробник · Інженер · Тімлід | Ops · Асистент · Менеджер |

→ [Повне порівняння Cowork та Code](https://github.com/FlorianBruniaux/claude-cowork-guide/blob/main/reference/comparison.md)

### Гід для прийняття рішень

- **Пишете документ або проводите дослідження?** → Claude AI
- **Програмування: рефакторинг, дебаг, тести?** → Claude Code (ви в правильному місці)
- **Організація файлів, обробка PDF, без коду?** → Cowork

## Сценарії використання

- **Організація файлів** — Безлад у папках → впорядкована структура.
- **Відстеження витрат** — Чеки/квитанції → звіти в Excel.
- **Синтез звітів** — Декілька документів → єдиний підсумковий звіт.
- **Підготовка до зустрічей** — Дослідження → брифінг-документи.

→ [Детальні робочі процеси](https://github.com/FlorianBruniaux/claude-cowork-guide/tree/main/workflows)

## Огляд безпеки

Офіційної документації з безпеки поки що не існує. Основні практики:

1. **Виділений робочий простір** — Ніколи не надавайте доступ до "Документів" або "Робочого столу" цілком.
2. **Перегляд планів** — Перевіряйте кожну дію перед схваленням.
3. **Жодних паролів** — Тримайте конфіденційні дані (ключі, паролі) за межами робочого простору.
4. **Бекап перш за все** — Робіть копії перед деструктивними операціями.

→ [Повний гайд із безпеки](https://github.com/FlorianBruniaux/claude-cowork-guide/blob/main/guide/03-security.md)
