# Claude Cowork: Агентний десктоп для інтелектуальної праці

> **📦 Повна документація перенесена в окремий репозиторій**

## Швидкий огляд

**Cowork** — це функція агентного десктопа Claude, яка надає автономні можливості AI нетехнічним користувачам через додаток Claude Desktop. Замість команд у терміналі, Cowork працює безпосередньо з локальними папками та файлами.

### Основні факти

| Аспект | Деталі |
|--------|---------|
| **Статус** | Research preview (Січень 2026) |
| **Доступ** | Підписка Pro ($20/міс) або Max ($100-200/міс), тільки для macOS |
| **Фокус** | Маніпуляція файлами, генерація документів, організація |
| **Відмінність від Code** | Без виконання коду — робота тільки з файлами |

---

## Три інструменти Claude: Який обрати вам?

Три інструменти, одна підписка ($20/міс Pro). Вони доповнюють один одного, а не конкурують.

| | Claude AI | Claude Code | Cowork |
|---|-----------|-------------|--------|
| **Інтерфейс** | Веб / Мобільний | Термінал (CLI) | Десктопний додаток |
| **Слоган** | Ви пишете, думаєте, шукаєте | Ви програмуєте масштабно | Ви автоматизуєте без коду |
| **Визначення** | Універсальний розмовний асистент | Автономний агент для цілих кодобаз | Агентні воркфлоу для файлів (не для девів) |
| **Основні кейси** | Тексти, брейнштормінг, дослідження | Налагодження, рефакторинг, тести | Організація файлів, витяг даних з PDF |
| **Виконує код** | Ні | Так | Ні |
| **Доступ до файлів** | Тільки завантаження | Повний | Пісочниця (папка) |
| **Налаштування** | Не потрібне | npm install -g @anthropic-ai/claude-code | Встановлення додатка macOS |
| **Зрілість** | Production | Production | Research preview |
| **Ідеальний профіль** | Письменник · Консультант · Студент | Розробник · Інженер · Техлід | Ops · Асистент · SMB (не-тех) |
| **Головний компроміс** | Немає доступу до системи | Витрати токенів у великих проектах | Тільки macOS, обмежений конфіг |

→ [Повне порівняння Cowork vs Code](https://github.com/FlorianBruniaux/claude-cowork-guide/blob/main/reference/comparison.md)

### Посібник з вибору

- **Пишете документ або проводите дослідження?** → Claude AI
- **Програмуєте: рефакторинг, налагодження, тести?** → Claude Code (ви в правильному місці)
- **Організовуєте файли, витягуєте дані з PDF, без коду?** → Cowork

## Кейси використання

- **Організація файлів** — Безлад у папках → організована структура.
- **Відстеження витрат** — Чеки → звіти в Excel.
- **Синтез звітів** — Кілька документів → єдиний звіт.
- **Підготовка до зустрічей** — Дослідження → брифінг-документи.

→ [Детальні воркфлоу](https://github.com/FlorianBruniaux/claude-cowork-guide/tree/main/workflows)

## Безпека (стисло)

Офіційної документації з безпеки поки не існує. Основні практики:

1. **Виділений робочий простір** — Ніколи не надавайте доступ до папок Документи/Робочий стіл.
2. **Перегляд планів** — Перевіряйте кожну дію перед схваленням.
3. **Жодних паролів** — Тримайте конфіденційні дані поза робочим простором.
4. **Бекап перш за все** — Робіть резервні копії перед деструктивними операціями.

→ [Повний посібник з безпеки](https://github.com/FlorianBruniaux/claude-cowork-guide/blob/main/guide/03-security.md)
