# Сторонні інструменти для Claude Code

> **Остання перевірка**: Квітень 2026

Ця сторінка містить каталог інструментів, створених спільнотою для розширення можливостей Claude Code. Усі інструменти були перевірені за їхніми публічними репозиторіями.

---

## 1. Відстеження токенів та витрат

- **ccusage**: Найпопулярніший інструмент для моніторингу витрат. Надає щоденні, щомісячні та сесійні звіти. Підтримує віджети macOS та розширення Raycast.
- **ccburn**: Python-утиліта, яка малює красиві графіки споживання токенів прямо в терміналі.
- **RTK (Rust Token Killer)**: Проксі-CLI, який фільтрує вивід команд (тести, логі git) перед тим, як вони потраплять до Claude. Зменшує споживання токенів на 50-90%.

---

## 2. Управління сесіями та контролем

- **Entire CLI**: Платформа для професійного управління сесіями ШІ. Дозволяє створювати чекпоінти, "відмотувати" сесію назад та забезпечує аудит дій агента (SOC2/HIPAA ready).
- **claude-code-viewer**: Веб-інтерфейс для зручного перегляду історії ваших чатів Claude Code з підсвіткою синтаксису.

---

## 3. Безпека та якість конфігурації

- **AgentShield**: Сканер безпеки для вашої директорії `.claude/`. Перевіряє наявність секретів, небезпечних хуків та надмірних прав доступу за 100+ правилами.
- **Caliber**: Оцінює якість вашого файлу `CLAUDE.md` за 100-бальною шкалою та виявляє "дрейф" (коли код змінився, а документація для ШІ застаріла).

---

## 4. Карта проекту та стандарти

- **codesight**: Використовує AST-аналіз для створення структурованої карти проекту (маршрути, схеми БД, залежності). Економить токени на початковому дослідженні коду в 10 разів.
- **Packmind**: Платформа "ContextOps" для синхронізації стандартів розробки між багатьма репозиторіями та різними агентами (Claude, Cursor, Copilot).

---

## 5. Мульти-агентна оркестрація

- **abtop**: TUI-панель (у стилі htop), яка показує всі активні сесії Claude Code, їх споживання токенів, ліміти та дерево субагентів.
- **Ruflo (раніше claude-flow)**: Найпопулярніший фреймворк для запуску цілих "роїв" (swarms) агентів для складних завдань.
- **Conductor**: macOS додаток для паралельного запуску кількох Claude Code у різних git-гілках з інтегрованим переглядом diff-ів.

---

## 6. Екосистема плагінів та скілів

- **Superpowers**: Потужний набір скілів для професійної розробки (мозковий штурм, планування, TDD-розробка, код-рев'ю).
- **gstack**: Скіл-пак від CEO Y Combinator (Garry Tan) для повного циклу розробки: від планування до деплою.

---

## Рекомендації за ролями

| Роль | Рекомендований стек |
|---------|-------------------|
| **Solo-розробник** | ccusage + claude-code-viewer |
| **Команда (2-5 осіб)** | ccusage + Conductor або multiclaude |
| **Ентерпрайз** | ccusage (MCP) + AgentShield + Entire CLI |
| **Power User** | abtop + RTK + codesight |
