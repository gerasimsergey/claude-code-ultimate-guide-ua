# Екосистема MCP-серверів

**Останнє оновлення**: Травень 2026 • **Наступний перегляд**: Червень 2026

Цей посібник охоплює перевірені MCP-сервери спільноти, окрім офіційних серверів Anthropic. Усі перелічені сервери були оцінені за критеріями готовності до продакшну, активності розробки та безпеки.

> **Не впевнені, що обрати: MCP-сервер чи CLI-інструмент?** Дивіться [Посібник з прийняття рішень MCP vs CLI](./mcp-vs-cli.md).

---

## Офіційні vs Спільнотні сервери

| Тип | Приклади | Характеристики | Коли використовувати |
|------|----------|-----------------|----------|
| **Офіційні** | filesystem, memory, github | Підтримка Anthropic, гарантована стабільність | Вибір за замовчуванням |
| **Спільнотні** | Playwright, Semgrep, Kubernetes | Підтримка організацій/осіб, готові до продакшну | Спеціалізовані потреби |

---

## Еволюція екосистеми (2026)

- **Стандартизація Linux Foundation**: MCP став офіційним стандартом під егідою Agentic AI Foundation.
- **Формат MCPB**: Стандартизований формат пакетів для встановлення серверів в один клік.
- **MCP Apps**: Нова специфікація для інтерактивних інструментів (Slack, Figma, Asana).

## Валідовані сервери

### 1. Автоматизація браузера

- **Playwright MCP (Microsoft)**: Офіційний сервер для автоматизації, оптимізований для LLM. Використовує дерева доступності (accessibility trees) замість скріншотів, що значно економить токени.
- **Browserbase MCP**: Хмарна автоматизація з підтримкою "стелс-режиму" та проксі для складних завдань.
- **Chrome DevTools MCP**: Дозволяє Claude інспектувати консоль браузера та мережеві запити в реальному часі.

### 2. DevOps та інфраструктура

- **Kubernetes MCP (Red Hat)**: "Kubectl людською мовою". Дозволяє переглядати логи, масштабувати деплойменти та керувати ресурсами кластера через чат.
- **Sentry MCP**: Офіційний сервер для діагностики помилок. Claude зчитує стек-трейс помилки, знаходить причину в коді та пропонує виправлення.
- **Vercel MCP**: Повний цикл CI/CD: створення проектів, керування змінними оточення та моніторинг деплоїв.

### 3. Безпека та аналіз коду

- **Semgrep MCP**: Офіційний сервер для сканування вразливостей (SAST). Claude автоматично перевіряє написаний код на наявність дірок у безпеці (SQLi, XSS) перед комітом.

### 4. Пошук коду та документація

- **Grepai MCP**: Семантичний пошук коду через локальні ембеддінги (Ollama). Дозволяє шукати за наміром (наприклад, "де тут логіка оплати?"), а не лише за точним збігом тексту.
- **Context7 MCP**: Надає актуальну документацію для понад 500 бібліотек (LangChain, SDK тощо), усуваючи галюцинації Claude щодо API.

### 5. Управління проектами

- **Linear MCP**: Автоматичне створення тікетів, оновлення статусів та коментування завдань у Linear.

## Патерн довідкових файлів (Reference File Pattern)

Для стабільної роботи з MCP-серверами (особливо складними, як Sentry чи Datadog), рекомендується використовувати довідкові файли:
1. Створіть файл `.claude/skills/my-skill/references/mcp-name.md`.
2. Опишіть там специфічний синтаксис запитів та відомі помилки.
3. Першим кроком у `SKILL.md` змусьте Claude прочитати цей довідник.

Це усуває цикли повторних спроб через неправильний синтаксис і робить роботу агента передбачуваною.
